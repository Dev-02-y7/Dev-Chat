Dev Chat

Dev Chat is a real-time private messaging application inspired by modern chat platforms like WhatsApp and Discord. I built this project to learn how real-time systems work and how frontend and backend applications communicate using WebSockets.<img width="509" height="256" alt="Screenshot 2026-05-25 104320" src="https://github.com/user-attachments/assets/a4f8b198-cd4b-43a4-b9c1-3218387b0128" />


Features -

- Real-time private messaging using Socket.IO and WebSockets
- Live online users system with instant chat updates
- One-to-one conversations inspired by modern chat platforms
- Typing indicators and unread message notifications
- Responsive and clean WhatsApp-style user interface
- Event-driven frontend and backend communication using Node.js and Express.js

Built using -

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js
- Socket.IO

What I Learned -

This project helped me gain hands-on experience with:

- Real-time communication using WebSockets
- Socket.IO event handling
- Frontend and backend integration
- Managing conversation state
- Event-driven architecture
- Debugging live communication systems
- Structuring larger full-stack projects

Getting Started -

Clone the repository:

```bash
git clone https://github.com/your-username/dev-chat.git
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node server.js
```

Open:

```txt
http://localhost:3000
```
Why I Built This -

I wanted to challenge myself with something more advanced than a basic frontend project. Building a real-time chat application helped me understand how modern communication systems work behind the scenes and improved both my frontend and backend development skills.
